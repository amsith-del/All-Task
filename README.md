 Portfolio Website

## Overview
This project is a multi-page portfolio website created using pure HTML only.  
The website includes Home, About, Projects, and Contact pages with semantic HTML structure and built-in HTML presentation attributes.

--
## Pages Included

- Home Page
- About Page
- Projects Page
- Contact Page
---
## Features

- Semantic HTML structure
- Navigation menu on all pages
- Hero section
- Skills table
- Project showcase section
- Contact form
- External project links
- Favicon support
- Pure HTML implementation without CSS or JavaScript
---
## Technologies Used
- HTML5
